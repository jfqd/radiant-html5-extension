h1. Radiant html5 extension

This extension adds html5 radius tags to "radiant":http://github.com/radiant/radiant.

h2. Installation

Starting from your radiant application root directory:

<pre><code>git clone git://github.com/jfqd/radiant-html5-extension.git vendor/extensions/html5
rake production radiant:extensions:html5:update</code></pre>

h2. About

This extension adds a few radius tags to support new html5 elements.

| <pre><r:html5:doctype /></pre>         | renders new html5 doctype             |
| <pre><r:html5:charset /></pre>         | renders an utf8-encoding meta charset |
| <pre><r:html5:legacy_support /></pre>  | renders a script element to load modernizr js library to support html5 for older browsers |
| <pre><r:html5:article /></pre>         | renders new html5 article element     |
| <pre><r:html5:section /></pre>         | renders new html5 section element     |
| <pre><r:html5:nav /></pre>             | renders new html5 nav element         |
| <pre><r:html5:aside /></pre>           | renders new html5 aside element       |
| <pre><r:html5:hgroup /></pre>          | renders new html5 hgroup element      |
| <pre><r:html5:header /></pre>          | renders new html5 header element      |
| <pre><r:html5:footer /></pre>          | renders new html5 footer element      |

h2. Samples

<pre><code><r:html5:article class="first" data-id="34">

h1. Article Headline

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

</r:html5:article></code></pre>

Output:

<pre><code><article class="first" data-id="34">
  <h1>Article Headline</h1>
  <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
</article></code></pre>
